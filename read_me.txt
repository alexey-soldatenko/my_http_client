Данное приложение представляет собой простой текстовый http-клиент.
В ходе его создания были написаны библиотеки для создания http-запросов,
отправки их на сервер, получение ответа (пакет query).
Была написана библиотека для парсинга html-страниц, создания dom-дерева (пакет 
dom_tree). 
Также была написана библиотека для парсинга css-файла (пакет css_tree).
Для создания дерева отображения также используется оригинальная библиотека (пакет
render_tree).
Разумеется, есть куча других библиотек для этих целей. Но...
Для визуального отображения используется библиотека PyQt5. Конечно, Qt предоставляет
хорошие инструменты для web-просмотра, такие как WebKit, но целью данного проекта
было разобраться во внутренней работе браузера, путём создания, так скажем его
аналога, а не использования уже готового. Поэтому использовался минимальный 
функционал библиотеки (пакет win_qt).

Запуск приложения осуществляется путём запуска http_client.py
